# Imágenes de Actividades del Gimnasio

Este directorio debe contener las imágenes de las actividades del gimnasio que se mostrarán en el carousel de la página principal.

## Imágenes Requeridas

Coloca las siguientes imágenes en este directorio:

1. **functional.jpg** - Entrenamiento Funcional
   - Sugerencia: Personas realizando ejercicios funcionales con TRX, kettlebells, etc.

2. **weights.jpg** - Musculación
   - Sugerencia: Personas entrenando con pesas, mancuernas o máquinas

3. **cardio.jpg** - Cardio Intensivo
   - Sugerencia: Clase de cardio en grupo, spinning, cinta de correr, etc.

4. **yoga.jpg** - Yoga y Estiramiento
   - Sugerencia: Sesión de yoga o pilates en grupo

5. **spinning.jpg** - Spinning
   - Sugerencia: Clase de spinning o ciclismo indoor

6. **crossfit.jpg** - CrossFit
   - Sugerencia: Entrenamiento de CrossFit con box jumps, wall balls, etc.

## Especificaciones de las Imágenes

- **Formato**: JPG o WebP
- **Dimensiones recomendadas**: 1200x800px (ratio 3:2) o 1600x900px (ratio 16:9)
- **Peso máximo**: 200KB por imagen (para optimizar carga)
- **Calidad**: Alta resolución pero comprimidas para web

## Placeholder Temporal

Mientras no tengas las imágenes reales, puedes:
- Usar imágenes de stock de sitios gratuitos como Unsplash, Pexels o Pixabay
- Buscar con términos: "gym workout", "functional training", "crossfit", "yoga class", etc.
- Usar herramientas como Squoosh.app para optimizar el peso de las imágenes

## Ejemplo de URLs de Placeholder

Si quieres usar placeholders temporales de internet, puedes modificar el componente
`GymActivitiesCarousel.tsx` para usar URLs como:
- `https://images.unsplash.com/photo-[id]?w=1200&q=80`
